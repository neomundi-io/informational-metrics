# Informational Metrics

Public methodological notes on NeoMundi informational observability metrics for AI runtime governance.

This repository documents public conceptual definitions related to informational metrics, including:

- volumetry;
- volumetric density;
- informational energy;
- informational density;
- methodological boundaries of interpretation.

This repository does not disclose proprietary internal implementations, production coefficients, optimization parameters, thresholds, weights, or runtime inference mechanisms used in the NeoMundi engine.

## Methodological Position

These metrics are not absolute judgments of truth, quality, or compliance.

They are informational observability artifacts intended to contextualize the behavior of an AI generation during execution.

They should be interpreted alongside other signals, including runtime stability signals, governance decisions, and business context.
